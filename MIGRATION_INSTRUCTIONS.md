# הוראות מיגרציה

## שינויים במסד הנתונים

הוספתי:
1. טבלת `TimeSlot` לניהול תורים
2. שדה `slotDuration` ב-`Business`

## כדי להריץ את המיגרציה:

```bash
# עצור את השרת (Ctrl+C)

# הרץ מיגרציה
npx prisma migrate dev --name add_timeslots

# אם זה מבקש שם, הקלד: add_timeslots

# הפעל שוב את השרת
npm run dev
```

## מה השתנה:

### 1. ✅ תוקן התפריט הצדדי
- עכשיו רואים את התפריט
- צבעים מעוצבים

### 2. ✅ כפתורי התחברות מוסתרים כשמחובר
- רק משתמשים מחוברים רואים את פרטיהם

### 3. ✅ חיפוש משופר
- חיפוש לפי קטגוריות (כפתורים גדולים)
- חיפוש לפי עיר (תגיות)
- חיפוש טקסט חופשי

### 4. 🔄 מערכת תורים חדשה (TimeSlot)
- בעל עסק יוכל לפתוח תורים
- לקוחות יבחרו מתוך התורים הפנויים
- כל תור = תאריך + שעת התחלה + שעת סיום

### 5. 🔄 שליחת מייל
- API מוכן
- צריך אינטגרציה עם שירות מייל (Resend/SendGrid)

## הבא:
לאחר המיגרציה, צריך לבנות:
1. דף ליצירת תורים לבעל עסק
2. דף לבחירת תורים ללקוחות
3. אינטגרציה של שליחת מייל
